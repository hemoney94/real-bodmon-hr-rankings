REAL BODMON HR RANKINGS V3

This version includes a Netlify Function proxy so Safari does not call Ballpark Pal directly.

DEPLOYMENT:
1. Put these files in a GitHub repository, preserving the folders.
2. In Netlify choose Add new site > Import an existing project.
3. Select the repository and deploy. No build command is required.
4. Open the Netlify URL, enter the Ballpark Pal API key, and load rankings.

Important: A simple drag-and-drop of index.html alone will NOT deploy the proxy function.
